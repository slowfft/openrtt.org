  • New version of the Real-time text Java open source software client SIPCon1 available

    Submitted by arnoud_R3TF on Wed, 09/17/2008 - 00:00

    SIPCon1, a Real-Time Text Java open source software client created by Omnitor and Trace is the first Open Source client using RFC4103 ToIP Real-Time Text.

    Now there is a new version of the Real-time text  SIPCon1 is available at
    http://sourceforge.net/projects/tipcon1/

    The version 1.4.70 and is now considered much better starting point for interoperability testing.
